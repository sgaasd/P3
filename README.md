# P3 Project
Robotics spring 2020 
3th semester project 

## Equipment 
1. Arduino MEGA
2. CrustCrawler Robot 3DOF
    - Motor ID configuration 
        * Joint 1 -> 0x01
        * Joint 2 -> 0x02
        * Joint 3 -> 0x03
        * Gripper left -> 0x04
            - Shadow ID -> 0x0A
        * Gripper right -> 0x05
            - Shadow ID -> 0x0A

## Steps
1. Clone repository
2. Verify and upload \P3\P3.ino to Arduino mega
